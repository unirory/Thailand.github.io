### Key CSS Properties Explained - **`background-size: cover;`**: This makes sure the background image covers the entire viewport, scaling it as needed. - **`background-position: center;`**: This centers the image in the viewport. - **`background-repeat: no-repeat;`**: Ensures the background image doesn't tile if it's smaller than the viewport. - **`height: 100vh;`**: Sets the height of the body to the full height of the viewport. - **`margin: 0;`**: Removes default body margin, ensuring the background image fills the entire screen.
